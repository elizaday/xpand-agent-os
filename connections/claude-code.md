# Claude Code Connection

Claude Code is one of Elizabeth's primary build tools for applications and interactive artifacts.

## Use

- Application and artifact building.
- Project-specific coding work.
- Specs should be tightened before long build sessions because credits can become a bottleneck.

## Translation Rule

For Claude Code projects, create or update `CLAUDE.md` inside the relevant project repo instead of copying this whole context folder.

## Docs Rule

For current Claude Code behavior or setup, check current official Anthropic documentation before answering.
