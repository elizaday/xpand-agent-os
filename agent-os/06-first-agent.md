# First Agent

Recommended first agent: Project Triage Agent.

## Job

Given messy current work, sort it by role, urgency, energy type, leverage, and meaning. Identify what to do this week, what to park, what to kill, and what needs verification.

## Why This First

It uses Elizabeth's identity, context, preferences, decision filters, and memory without requiring risky access to email, billing, calendars, or client systems.

## Definition Of Done

- Agent has its own file in `agents/project-triage-agent.md`.
- Agent can triage a real list of projects.
- Output includes next actions, parked items, killed items, and verification needs.
- Elizabeth can tell whether the recommendation sounds like her priorities.
