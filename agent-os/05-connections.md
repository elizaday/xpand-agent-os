# Connections

Connections are external systems the OS can use or reason about.

## Current Connection Notes

- `connections/codex.md`: Codex setup and usage.
- `connections/claude-code.md`: Claude Code setup and usage.
- `connections/openclaw.md`: OpenClaw setup and status.
- `connections/monday.md`: Monday.com project board.
- `connections/github.md`: repositories and git workflow.
- `connections/netlify.md`: Netlify hosting notes.
- `connections/supabase.md`: Supabase notes for agentic prototypes.

## Rule

Before touching external systems, confirm the target, permissions, and risk. Do not automate paid, public, or client-facing actions without explicit user direction.
