# Elizabeth Codex Context

This is Elizabeth's master context project for Codex.

It is not tied to a single codebase. It is the source of truth for how Codex should understand Elizabeth's work, voice, priorities, and current project landscape.

Open this folder when you want Codex to help with:

- strategic planning
- writing in Elizabeth's voice
- Xpand Studio development
- project pruning
- creating project-specific Codex instructions
- turning messy context into usable artifacts

For a specific repo, create a small `AGENTS.md` inside that repo instead of copying this entire folder.
